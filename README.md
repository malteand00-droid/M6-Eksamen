Dette projekt er en hjemmeside om en fiktiv sportsklub, Aalborg Ultimate Frissbee klub. 
Det er udarbejdet af Steffen Pedersen og Malte Andersen, og har til formål at oversætte teoretiske metoder og diagrammer til et fiktivt praksis eksempel.
Hjemmesidens formål er at have diverse informationer om klubben og muliggøre tilmelding af nye medlemmer.

Til projektet er der anvendt GitHub (desktop og browser) til repository og Git funktionen i Visual Studio Code, til udarbejdelse af kode. Dette er gjort da det var nemmest ift. opgavebeskrivelse og vi allerede har anvendt Visual Studio Code på tidligere semestre.



Der anvendes GitHub projects til opgavestyring vha. Kanban. (Argumenter for kanban i stedet for scrum)
