# Aalborg Ultimate Frisbee Klub - Hjemmeside

Dette projekt er en hjemmeside for den fiktive sportsklub **Aalborg Ultimate Frisbee Klub (AUFK)**. 

## Projektets Formål

Hjemmesiden skal fungere som klubbens ansigt udadtil og give brugere:
- Information om sporten Ultimate Frisbee
- Holdoversigter og medlemmer
- Træningsplan og kampprogram
- Mulighed for at blive medlem
- Kontaktinformation
